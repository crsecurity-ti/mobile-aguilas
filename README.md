# AguilasSeguridad Mobile

React Native + Expo app for Aguilas Seguridad.

## Setup

```bash
npm install
npx expo start
```

## Build (EAS)

```bash
# QA build
eas build --profile qa --platform android

# Production build
eas build --profile production --platform android
```

## Environment

Copy `.env.example` to `.env` and fill in the values.
